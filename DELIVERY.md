## 二、App 概况

**名称**：研芽 · 30天语文备考计划
**风格**：温馨可爱 × 高级感（蜜桃粉 + 薰衣草紫渐变、玻璃拟态卡片、柔和阴影）
**字体**：Quicksand + 思源黑体
**数据**：全部保存在手机本地（localStorage），不上传任何服务器。

**功能一览**
- 🏠 **首页 Dashboard**：今日进度环、连续打卡（🔥）、距收官倒计时、已搞定天数、番茄数、每日一句鼓励、当日验收卡片、快捷入口
- 📅 **30 天计划**：5 个阶段分组，点击任意一天打开详情弹层（333 / 826 / 英语政治 三段任务 + 当日验收 + 学习提示），可标记当日完成
- ✅ **今日打卡**：按每日时间表 13 个时段逐项勾选，进度条实时变化，全部完成自动撒花 + 给一个随机小奖励
- 📝 **复盘**：三个标签页 — 每周自查清单（5 周 × 8 项点格子）/ 周日轻量复盘模板 / 五周周报自由填写
- 🍅 **番茄钟**：25 分钟专注 + 5 分钟休息，环形倒计时，完成累计统计
- 📔 **心情日记**：每日 5 档心情 + 文字记录，可回看历史
- 🌷 **我的**：学习方法库（6 种关键方法展开）/ 每日时间表 / 四科目标 / 8 月强化衔接 / 资料来源 / 关于 / 重置

## 三、给女朋友的安装方式（最简单）

### 方式 A：在线访问（推荐，零配置）
1. 把 `sweet-planner` 目录放到任意静态托管空间（GitHub Pages、Vercel、Netlify、或本地).
2. 安卓手机用 Chrome 打开网址 → 菜单 → **"添加到主屏幕"** → 桌面会出现"研芽"图标，点击即全屏运行，离线也能用。

### 方式 B：纯本地（不联网）
1. 把整个 `sweet-planner` 文件夹拷到手机存储（或用邮箱/QQ发给女友保存到手机）。
2. 安卓可装一个轻量本地服务器（如 "KSWEB" 或 "Simple HTTP Server"），把 `sweet-planner` 设为根目录，浏览器访问 `localhost:端口` 后"添加到主屏幕"。或直接用 Chrome 打开 `index.html` 也可正常浏览（PWA 离线缓存需 http(s) 才生效，文件协议下打卡等仍可用）。

### 方式 C：你在电脑演示
本目录直接运行：
```bash
cd "C:\Users\yusheng\ZCodeProject\sweet-planner"
python -m http.server 8848
# 浏览器打开 http://localhost:8848/index.html
```

## 四、二次优化要点（相对原"优化优化版"）

1. **时间线统一**：30 天完整日历对齐 2026-07-06 至 2026-08-04，避免周日复盘节点错位。
2. **每日表述精修**：把"学/看"型动词改为可交付动词（画框架、默写、做提纲、试写简案）。
3. **心态缓冲**：新增"节奏 > 激情""只写事实和下一步、不自责""低电量日也算策略"等提示。
4. **App 化**：内容结构化后端到端打包成可直接触达的打卡 App，纸质计划变成随身伴行工具。
