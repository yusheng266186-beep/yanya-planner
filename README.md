# 研芽 · 30天语文备考计划 🌸

西华师范大学 · 学科教学（语文）30 天基础夯实计划 — 温馨可爱风 PWA。

## 📲 在手机上使用

1. 安卓用 **Chrome** 打开： <https://yusheng266186-beep.github.io/yanya-planner/>
2. 菜单 → **添加到主屏幕** → 桌面出现「研芽」图标，点开即全屏运行，离线也可用。

## ✨ 功能

- 首页 Dashboard：今日进度环、连续打卡、倒计时、每日鼓励
- 30 天详细计划（5 组，含每日验收与提示，可标记当日完成）
- 今日打卡（13 时段逐项勾选，全完成自动撒花 + 随机小奖励）
- 周复盘：每周自查清单 / 周日轻量复盘模板 / 五周周报
- 番茄钟（25' 专注 + 5' 休息）
- 心情日记（5 档心情 + 文字）
- 学习方法库、每日时间表、四科目标、8 月衔接、资料来源

数据全部保存在本机，不上传任何服务器。🌷
